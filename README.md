# Sales Dashboard | Tableau

This project is an interactive Tableau Sales Dashboard built to analyze yearly sales performance, profit, quantity, sub-category comparison, and weekly trends. The dashboard compares the selected year with the previous year and highlights key business performance indicators.

This dashboard was created as a beginner practice project while following a Tableau tutorial.

## Project Overview

The dashboard provides a clear overview of sales performance for 2023 compared with 2022. It includes KPI cards, trend analysis, sub-category performance, profit/loss visualization, and highest/lowest month indicators.

## Dashboard Features

- Year-over-year comparison for Sales, Profit, and Quantity
- KPI cards for:
  - Total Sales
  - Total Profit
  - Total Quantity
- Highest and lowest month indicators
- Sub-category comparison by sales and profit
- Weekly sales and profit trend analysis
- Profit and loss highlighting
- Interactive filters for category, state, region, city, sub-category, and week
- Clean dashboard layout designed for business reporting

## Tools Used

- Tableau Public / Tableau Desktop
- Sales dataset
- Calculated fields
- Parameters
- Dual-axis charts
- KPI cards
- Interactive dashboard filters

## Key Insights

- Total Sales reached approximately $733K in 2023.
- Total Profit reached approximately $93K.
- Total Quantity reached approximately 12K units.
- Sales increased by 20.4% compared with the previous year.
- Profit increased by 14.2% compared with the previous year.
- Quantity increased by 27% compared with the previous year.

## Project Goal

The goal of this project is to practice Tableau dashboard development and build a professional data analytics portfolio project using business intelligence techniques, KPI reporting, and visual storytelling.
